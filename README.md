# Kinaipor

shadcn/ui + Cloudflare Workers 專案

## 技術棧

- React 18
- TypeScript
- Vite
- shadcn/ui
- Tailwind CSS
- Cloudflare Workers
- Hono

## 快速開始

```bash
# 安裝依賴
npm install

# 本地開發
npm run dev

# 構建
npm run build

# 部署到 Cloudflare
npm run deploy
```

## 部署

使用 Cloudflare Wrangler 部署：

```bash
wrangler login
wrangler deploy
```
